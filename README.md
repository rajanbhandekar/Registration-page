# Registration and Login Page

A clean, responsive, and user-friendly web interface for user authentication, featuring dedicated pages for email registration and secure login.

## 🚀 Features
* **Responsive Design:** Works flawlessly on mobile, tablet, and desktop screens.
* **Form Validation:** Client-side validation for emails and strong password requirements.
* **User-Friendly UI:** Modern styling with clean, accessible input fields.

## 📁 Project Structure
The repository consists of the following primary files:
* `emailRegistration.html` - The sign-up form interface where new users create an account.
* `emailLogin.html` - The login portal for existing users to authenticate.

## 🛠️ Tech Stack
* **HTML5:** Structuring the authentication forms.
* **CSS3:** Custom styling and responsive layout configurations.
* **JavaScript (Optional):** Client-side validation logic.

## 💻 How to Run the Project Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```

2. **Navigate to the project folder:**
   ```bash
   cd Registration-page
   ```

3. **Open the files:**
   Simply double-click `emailRegistration.html` or `emailLogin.html` to open and view them in any web browser.
